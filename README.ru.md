# 🛡️ Amnezia Web Panel

**Веб-панель управления для AmneziaVPN** — установка и управление VPN-протоколами на собственных серверах через SSH.

**Языки:** [🇮🇷 فارسی](README.fa.md) | 🇷🇺 Русский | [🇨🇳 中文](README.zh.md) | [🇬🇧 English](README.md)

---

## ✨ Возможности

| Протокол | Описание |
|---|---|
| 🔒 **AmneziaWG** | WireGuard с обфускацией против DPI |
| 🔑 **WireGuard** | Быстрый современный VPN |
| ⚡ **Xray VLESS Reality** | Маскировка под HTTPS, домен не нужен |
| 🌊 **VLESS WebSocket** | CDN-совместимый, WS+TLS |
| 📶 **VLESS gRPC** | CDN, мультиплексирование |
| 📡 **VMess WebSocket** | Широкая поддержка клиентов |
| 🐴 **Trojan** | Trojan через TLS |
| 🌑 **Shadowsocks 2022** | Лёгкая современная обфускация |
| 🌐 **OpenVPN** | Классический VPN |

### Возможности панели
- 🌍 **4 языка** — English, فارسی, Русский, 中文
- 📊 **Мониторинг трафика** — rx/tx на каждого пользователя
- 📅 **Дата истечения** — автоотключение при истечении срока
- 👥 **Массовое создание** — до 100 пользователей за раз
- 📤 **Экспорт CSV** — для биллинга и отчётов
- 🔍 **Обнаружение конфигов Amnezia** — находит конфиги из приложения Amnezia
- ✏️ **Редактирование** — имя, email, заметки, срок, лимит
- 📱 **QR-коды** — для мобильных клиентов
- 🔄 **Самообновление** — обновление с GitHub в один клик
- 🔥 **Автооткрытие портов** — при установке протокола

## 🚀 Установка одной командой

На сервере Ubuntu/Debian:

```bash
bash <(curl -s https://raw.githubusercontent.com/ScannerVpn/Amnezia-Web-Panel/main/install.sh)
```

После установки отобразятся учётные данные:

```
╔══════════════════════════════════════╗
║    Panel installed successfully! ✅   ║
╚══════════════════════════════════════╝

  Panel URL:   http://YOUR-IP:54325
  Username:    admin
  Password:    xxxxxxxxxxxx
```

## 📋 Требования

**Сервер панели:** Ubuntu 20.04+ или Debian 11+, 512MB RAM, интернет.

**VPN-серверы:** Ubuntu 20.04+ или Debian 11+, SSH root/sudo, интернет.

---

## 🏪 Функции для реселлеров

- **Лимиты трафика** — ГБ на пользователя с визуальным прогрессом
- **Дата истечения** — автоотключение в нужную дату
- **Массовое создание** — 5–100 пользователей с префиксом и нумерацией
- **Экспорт CSV** — все пользователи с данными об использовании
- **Заметки и email** — информация о клиенте
- **Поиск** — мгновенный поиск по таблице

## 🔄 Обновление

В настройках панели → кнопка **Update from GitHub**.

Или вручную:
```bash
cd /opt/amnezia-panel && git pull && docker compose up -d --build
```
