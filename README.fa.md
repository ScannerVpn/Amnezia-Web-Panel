# 🛡️ پنل مدیریت Amnezia

**پنل تحت وب برای مدیریت AmneziaVPN** — نصب و مدیریت پروتکل‌های VPN روی سرورهای شخصی از طریق SSH، با رابط کاربری چندزبانه و زیبا.

**زبان‌ها:** 🇮🇷 فارسی | [🇷🇺 Русский](README.ru.md) | [🇨🇳 中文](README.zh.md) | [🇬🇧 English](README.md)

---

## ✨ امکانات

| پروتکل | توضیحات |
|---|---|
| 🔒 **AmneziaWG** | WireGuard با obfuscation ضد DPI — مناسب ایران و چین |
| 🔑 **WireGuard** | VPN سریع و مدرن |
| ⚡ **Xray VLESS Reality** | ترافیک شبیه HTTPS — سخت‌ترین تشخیص، بدون نیاز به دامنه |
| 🌊 **VLESS WebSocket** | CDN-friendly، WS+TLS |
| 📶 **VLESS gRPC** | CDN-friendly، multiplexed |
| 📡 **VMess WebSocket** | پشتیبانی گسترده |
| 🐴 **Trojan** | Trojan روی TLS |
| 🌑 **Shadowsocks 2022** | obfuscation سبک و مدرن |
| 🌐 **OpenVPN** | VPN کلاسیک |

## 🚀 نصب تک‌دستوری

روی سرور Ubuntu/Debian اجرا کنید:

```bash
bash <(curl -s https://raw.githubusercontent.com/ScannerVpn/Amnezia-Web-Panel/main/install.sh)
```

بعد از نصب، اطلاعات ورود نمایش داده می‌شود:

```
╔══════════════════════════════════════╗
║    Panel installed successfully! ✅   ║
╚══════════════════════════════════════╝

  Panel URL:   http://YOUR-IP:54325
  Username:    admin
  Password:    xxxxxxxxxxxx
```

## 📋 پیش‌نیازها

**سرور پنل:**
- Ubuntu 20.04+ یا Debian 11+
- حداقل ۵۱۲MB RAM
- اتصال به اینترنت

**سرورهای VPN:**
- Ubuntu 20.04+ یا Debian 11+
- دسترسی SSH با root یا sudo
- اتصال به اینترنت

## 🏪 امکانات فروشندگان

این پنل برای فروش VPN طراحی شده است:

- **محدودیت ترافیک** — تنظیم حجم به ازای هر کاربر با نوار پیشرفت
- **تاریخ انقضا** — غیرفعال‌سازی خودکار در تاریخ مشخص
- **ایجاد دسته‌ای** — ساخت ۵ تا ۱۰۰ کاربر با پیشوند و شماره‌گذاری
- **خروجی CSV** — تمام کاربران با آمار مصرف
- **یادداشت و ایمیل** — اطلاعات مشتری برای هر کاربر
- **جستجو** — جستجوی زنده در جدول کاربران

## 🔄 به‌روزرسانی

از Settings پنل → دکمه **Update from GitHub**.

یا دستی:
```bash
cd /opt/amnezia-panel && git pull && docker compose up -d --build
```
